# React.js — Hands-On Practice

Two standalone React applications built with modern tooling, exploring component architecture and state management.

## Projects

### 1. `basic-react-app/`
A foundational React app covering:
- Functional components and JSX
- Props and component composition
- Modern project setup with **Vite** and **ESLint**

### 2. `react-state-class/`
State-focused React app covering:
- `useState` hook for local state management
- Re-rendering behaviour and controlled components
- Event handling in React

## Run Locally

```bash
# For either project
cd basic-react-app   # or react-state-class
npm install
npm run dev
```

Then open `http://localhost:5173`
